# Sala de Juegos (TP #1)
## Objetivo
Se debe realizar la aplicación “Sala de juegos”. La aplicación debe permitir a los usuarios medir sus capacidades cognitivas y motrices, debe ser simple de utilizar y permitir tener estadísticas de cada jugador y de cada juego. Los juegos deben guardar toda la información que sea importante para cada juego en particular. Solo se puede ingresar y jugar si es un usuario logueado o si se registra. Todos los juegos deben tener condiciones de finalización de victoria y de derrota claras.

## Tecnologías Utilizadas
* **Frontend**: Angular _(v21.2.10)_
* **Estilos**: Bootstrap _(v5.3.8)_

## Juegos Incluidos
1. **Ahorcado**
2. **Mayor o Menor**
3. **Preguntados**

## Contenidos Principales
* **Sistema de Autenticación**: Registro e inicio de sesión integrados.
* **Sala de Chat**: Comunicación en tiempo real entre todos los usuarios conectados.
* **Estadísticas y Resultados**: Mostrar estadísticas de cada jugador y de cada juego.
